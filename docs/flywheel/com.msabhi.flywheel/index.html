Middleware
Link copied to clipboard
common
typealias Middleware<S> = (Dispatch, GetState<S>) -> (Dispatch) -> Dispatch
Content copied to clipboard

Middleware helps to intercept actions before reaching the StateReserve reduce function. We can use Middleware to modify an received action, swallow the action to prevent reaching the reducer. Middleware by default runs on caller thread, which can be Main thread also. Though the concept of Middleware is similar to Redux. It is not recommended to use middleware for async operations. For that, we use the concept of SideEffects.
